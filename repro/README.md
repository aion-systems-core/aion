# AION Repro

Deterministic execution debugging for reproducible systems.

**Product documentation** (what it does, why it exists, CLI-only guide, examples) lives at the **repository root**: open `README.md` in the top-level **aion-systems-core** checkout, plus `RELEASE.md` and the `examples/` folder.

## Develop & test

From the repository root:

```bash
cargo test -p repro
```
