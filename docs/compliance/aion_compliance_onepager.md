# AION compliance one-pager

**AION = deterministic execution** — Same model, prompt, seed, and frozen runtime contract yields the same token stream and artefacts, so security and compliance teams can reason about *repeatable* AI behaviour.

**AION = audit trail** — Each run emits structured outputs (JSON, HTML, SVG) and a **capsule** suitable for retention, e-discovery, and change control.

**AION = evidence chain** — A linear chain of digests binds execution steps; verifiers can check **continuity** without trusting a single opaque blob.

**AION = replay symmetry** — **Replay** reconstructs the run and compares it to the persisted capsule, surfacing explicit symmetry flags and diffs.

**AION = policy enforcement** — **Governance** profiles constrain models, prompts, seeds, evidence presence, and replay outcomes; validation is machine-checkable.

**AION = model-agnostic** — The contract is about *recorded* runs and backends that honour the determinism envelope—not a single vendor SDK.
